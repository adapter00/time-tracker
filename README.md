
time tracking command for slack



## start

```
docker compose up -d
```

 start postgresql and slack cmd

## command

```
・start work
/tm start

・finish work
/tm stop [default now or yyyymm]

・start rest
/tm rstart 

・finish rest
/tm rstop 

・show month total duration
/tm show 


・show detail in month
/tm detail

・help
/tm help
```
